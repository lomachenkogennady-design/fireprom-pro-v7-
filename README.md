## 📦 Включённые модули

| Модуль | Описание |
|--------|----------|
| 🔩 **Калькулятор металла** | ОЦ, Х/К, Г/К — масса, резка, доставка, коэф. сложности |
| 📐 **DXF-парсер** | Загрузка и визуализация DXF-файлов с поддержкой дуг (bulge) |
| 🛒 **Корзина** | Сохранение позиций, экспорт CSV, печать, сохранение HTML |
| 🚪 **Калькулятор дверей** | Противопожарные, наружные, внутренние + опции |
| 📐 **Справочник просветов** | Нормативные размеры по СП/ГОСТ |
| 📄 **Генератор КП** | Коммерческое предложение с печатью/PDF |
| 🛠️ **Прайс монтажа** | 20+ позиций: замеры, доставка, демонтаж, монтаж, отделка |
| 📈 **RFM-анализ** | Сегментация клиентов: VIP / Активный / Спящий / Потерянный |
| 🏢 **CRM** | Клиенты, заявки, экспорт CSV |

## 🛠️ Технологии

- **HTML5** — семантическая разметка
- **CSS3** — CSS Variables, Grid, Flexbox, адаптивный дизайн
- **Vanilla JavaScript** — без фреймворков, чистый ES6+
- **LocalStorage** — автосохранение данных
- **PWA** — Splash screen, offline-режим, install prompt

## 📱 PWA-установка

1. Откройте сайт на смартфоне
2. Нажмите **«Поделиться»** → **«На экран "Домой»** (iOS) или **«Добавить на главный экран»** (Android)
3. Приложение установится как нативное
